Grønn koding vs. Ikke-grønn koding

Beskrivelse til grafen

Grafen viser forskjellen mellom grønn koding og vanlig koding. Grønn koding handler om å skrive kode som bruker mindre energi og jobber mer effektivt. Dette kan for eksempel være å laste inn bilder kun når de vises på skjermen (lazy loading), eller å huske tidligere beregninger slik at datamaskinen ikke må gjøre det samme flere ganger (caching). Slike teknikker er beskrevet i Python Docs og MDN Web Docs. Når vi skriver grønn kode, får vi raskere nettsider, mindre belastning på servere, og mindre strømforbruk. Det betyr ikke bare en bedre opplevelse for brukerne, men også at vi sparer miljøet. Ifølge Norges forskningsråd kan små endringer i måten vi bruker energi på ha stor betydning for å redusere klimagassutslipp. Det samme gjelder for koden vi skriver — effektiv kode er bra for både teknologien og planeten.

Python-eksempler

Nybegynner: Listeforståelse

Rask og effektiv måte å lage lister.

squares = [x**2 for x in range(10)]

Mellomnivå: Generatorer

Gir ett element om gangen og sparer minne.

def count_up_to(n):
  yield from range(n)

Avansert: Memoisering

Lagrer resultater for å unngå gjentatte beregninger.

from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
  return n if n < 2 else fib(n-1) + fib(n-2)

JavaScript-eksempler

Nybegynner: Lazy loading

Laster inn bilder kun når de trengs.

<img src="bilde.jpg" loading="lazy" alt="Et bilde">

Mellomnivå: Debounce-funksjon

Forhindrer at funksjoner kalles for ofte.

function debounce(func, delay) {
let timer;
return (...args) => {
  clearTimeout(timer);
  timer = setTimeout(() => func.apply(this, args), delay);
};
}

Avansert: Map og filter

Effektiv manipulering av arrays.

const numbers = [1, 2, 3, 4, 5];
const evenSquares = numbers.filter(n => n % 2 === 0).map(n => n * n);

Java-eksempler

Nybegynner: Enhanced for-loop

Mer lesbar iterasjon over lister.

for (int number : numbers) {
  System.out.println(number);
}

Mellomnivå: StringBuilder

Bygger strenger effektivt i løkker.

StringBuilder builder = new StringBuilder();
for (int i = 0; i < 100; i++) {
  builder.append(i);
}
String result = builder.toString();

Avansert: ParallelStream

Utnytter flere kjerner for raskere prosessering.

numbers.parallelStream().forEach(System.out::println);

C#-eksempler

Nybegynner: Foreach-løkke

Enkel iterasjon over lister.

foreach (var item in items)
{
  Console.WriteLine(item);
}

Mellomnivå: Using for ressurskontroll

Frigjør ressurser automatisk.

using (StreamReader reader = new StreamReader("file.txt"))
{
  var content = reader.ReadToEnd();
}

Avansert: Async nettverkskall

Utfører nettverkskall uten å blokkere tråder.

public async Task<string> GetDataAsync(string url)
{
  using var client = new HttpClient();
  return await client.GetStringAsync(url);
}
